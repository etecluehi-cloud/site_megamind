# Título do Projeto
Desenvolvimento de um Aplicativo e Site Educacional para Apoio à Preparação para o ENEM

# Objetivo
Desenvolver uma plataforma digital (aplicativo e site) voltada para estudantes da rede pública, com o objetivo de auxiliar na preparação para o ENEM, oferecendo conteúdos de Língua Portuguesa e Matemática de forma acessível, interativa e personalizada, promovendo maior inclusão educacional.

# Tecnologias Utilizadas
- Java (Android Studio)
- XML
- Firebase (Autenticação e Banco de Dados)
- HTML, CSS e Bootstrap
- GitHub

# Instruções de Execução

## Aplicativo
1. Abrir o projeto no Android Studio
2. Conectar um emulador ou dispositivo físico
3. Executar o projeto
4. Realizar cadastro ou login para acessar o sistema

## Site
1. Abrir a pasta do projeto
2. Executar o arquivo `index.html` em um navegador
3. Navegar pelas páginas disponíveis (algumas ainda em desenvolvimento)
